## 多因子分析框架（分析视角，可结合自身判断）

为每个时间框架综合评估方向。下列为各因子的常见解读倾向，供参考：

### 1. 因子解读倾向
- EMA 趋势：价格在 EMA 上方且均线多头排列 → 看涨；空头排列 → 看跌。
- MACD：线在零上且柱值正向扩大 → 看涨；零下且柱负向扩大 → 看跌。
- RSI：超卖反弹倾向 → 看涨；超买回调倾向 → 看跌；中间区间用 RSI 方向指示动量。
- 随机指标：超买 → 谨慎看跌；超卖 → 谨慎看涨；金叉/死叉跟随。
- 布林带：价格逼近下轨 → 支撑看涨；逼近上轨 → 压力看跌；带宽收缩预示变盘，结合其他因子。
- ADX：ADX>25 趋势强，+DI>-DI 看涨，反之看跌；ADX<20 震荡，信号弱化。
- OBV：OBV 趋势与价格同向确认；若价格新高而 OBV 未新高，背离看跌，反之亦然。
- VWAP：价格 > VWAP → 偏多，价格 < VWAP → 偏空。
- 支撑阻力 (60-bar 高低 / ATR 距离)：靠近支撑且远低于阻力 → 看涨倾向；靠近阻力 → 看跌倾向。
- Fibonacci 位置：在 0.382/0.5/0.618 反弹区获得支撑可看涨；若跌破关键 fib 转弱。
- 市场结构：出现 higher-highs/higher-lows (BOS) → 看涨；lower-highs/lower-lows (CHoCH) → 看跌。
- 动量 (5-bar ATR 归一化)：净正向动量强且 ATR 扩张 → 看涨持续；动量减弱或转负 → 看跌。
- 主动买卖量 (taker buy share)：当前主动买入占比高 → 看涨；持续走低 → 看跌。
- 相对成交量扩展：放量上涨 → 强看涨；放量下跌 → 强看跌；缩量反弹/下跌 → 信号弱。
- 资金费率 (funding_rate)：非常正（>0.05%）→ 多头拥挤，看跌；非常负 → 看涨（反指）。
- OI 与价格变化 (open_interest_change_pct vs price_change_24h_pct)：价涨+OI 涨 → 趋势延续看涨；价涨+OI 跌 → 弱势，可能反转；价跌+OI 涨 → 看跌；价跌+OI 跌 → 空头回补可能反弹。
- 多空账户比 (long_short_account_ratio)：散户大幅看多（>2）→ 反向看跌；极端看空 → 反向看涨。
- 主动买卖比率 (taker_buy_sell_ratio_5m)：>1 看涨，<1 看跌。
- PCR (put_call_ratio)：过高（>0.7）→ 恐慌底？看涨；过低 → 过度看涨可能回调。
- 期权偏斜 (skew_put_minus_call_iv)：正值=看跌期权更贵 → 避险/偏空；负值=看涨期权更贵 → 偏多。
- 最大痛点 (max_pain)：当前价格若低于 max pain，可能向上吸引；高于 max pain，可能向下吸引（重力效应）。
- 订单簿不平衡 (orderbook_imbalance_top50)：买盘墙厚（正值）→ 短期支撑看涨；卖盘墙厚（负值）→ 压力看跌。
- 杠杆拥挤度：若资金费率极端且持仓高，易发清算瀑布 → 反方向。
- FNG (fear_greed_index)：极度恐惧 → 看涨；极度贪婪 → 看跌。
- 市值与资金流：全市场市值上升 (total_mcap_change_24h_pct > 0) → 风险偏好回升，偏多；BTC 占比 (btc_dominance_pct) 快速下降 → 资金分流山寨。
- DXY 上涨、美债收益率上升 → 风险资产承压，看跌 BTC。
- MVRV：低 (<1) 价值低估 → 看涨；高 (>3) 过热 → 看跌。
- 内存池费用 (mempool_fastest_fee_sat_vb) 偏高：网络需求大，常伴链上活跃，中性偏多；极度拥堵可能阻碍使用。
- 哈希率 (hashrate_ehs) 趋势上升：网络安全信心增强，略偏多。
- 新闻：重大利好 → 看涨；重大利空 → 看跌。

- 方向：`composite > 0.05` → Long ▲；`< -0.05` → Short ▼；否则 Neutral ●。
- 信心 `confidence ∈ [0,1]`：综合方向强度 `|composite|`、各因子方向的一致性、以及可用数据的覆盖度三者评估（方向越强、因子越一致、数据越全 → 信心越高）。
